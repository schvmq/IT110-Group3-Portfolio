The Quote Collective

A showcase of our team's collaboration and a portal to our Quote Generator application.

About the Project

Our primary goal for this project was to design, build, and deploy a functional web application that interacts with a public API. We chose to create a Random Quote Generator using the DummyJSON Quotes API, a tool that provides inspirational and thought-provoking quotes on demand. This project demonstrates our skills in HTML, CSS, and JavaScript, focusing on clean UI/UX, effective API integration, and robust error handling. We utilized modern web development practices, including responsive design, to ensure a seamless experience across all devices.
